# HAIRC

Home Assistant IR Controller. 

## Overview

A Raspberry Pi Pico-based infrared controller designed for future integration with Home Assistant.

## Features

- ✅ IR transmit/receive functionality
- ✅ Command-line interface
- 🚧 Home Assistant integration (planned)

## Hardware Requirements

- Raspberry Pi Pico
- Adafruit IR LED trasceiver 

## Installation
Ensure that the pi pico has micropython firmware and install this library https://github.com/peterhinch/micropython_ir then the program will run.
